## 📌 Descripción

Este proyecto consiste en el desarrollo de una aplicación de escritorio en **Python** utilizando **Tkinter**, la cual permite implementar un **Árbol Binario de Búsqueda (ABB)** de manera gráfica e interactiva.

La aplicación fue desarrollada como parte del curso **Estructura de Datos** de la **Universidad Nacional Abierta y a Distancia (UNAD)**.

El sistema permite al usuario iniciar sesión, agregar nodos al árbol, realizar búsquedas, visualizar gráficamente la estructura del árbol y mostrar los recorridos clásicos de un Árbol Binario de Búsqueda.

---

# ✨ Características

- 🔐 Pantalla de inicio de sesión.
- 🌳 Inserción de nodos.
- 🔍 Búsqueda de nodos.
- 🎨 Visualización gráfica del árbol.
- 📈 Recorridos:
  - Preorden
  - Inorden
  - Posorden
- 🧹 Limpiar completamente el árbol.
- 🚫 Validación de datos.
- 🚫 Evita insertar nodos repetidos.
- 🚫 Limita el árbol a **4 niveles**.
- 🖥️ Interfaz gráfica moderna realizada con Tkinter.

# 📖 Funcionalidades

## Login

- Validación mediante contraseña.
- Pantalla moderna.
- Logo personalizado.

---

## Árbol Binario

Permite:

- Insertar nodos.
- Buscar valores.
- Dibujar automáticamente el árbol.
- Mostrar recorridos.

---

## Recorridos

La aplicación muestra automáticamente:

- ✔ Preorden
- ✔ Inorden
- ✔ Posorden

---

## Validaciones

El sistema controla:

- Campos vacíos.
- Solo números enteros.
- Nodos duplicados.
- Árbol máximo de 4 niveles.
